#  React Task Manager

A simple Task Manager application built with **React**.  
The app allows users to add tasks, mark them complete, filter them, delete them, also includes feedback and popups.  
All tasks persist in `localStorage`, so nothing disappears on refresh.

---

##  Features

###  Add Tasks  
- Allows user to add tasks to the list.

###  Complete Tasks  
- Allows user to check off tasks using a check box.


###  Delete Tasks  
- Allows user to delete tasks. 

###  Filters  
Filter tasks by:
- **All**
- **Active**
- **Completed**

###  Task Counter  
- Counter to display number of tasks on the list.

###  Clear Completed  
- Button clears all completed tasks from the list. 

###  Feedback Message  
- Shows short feedback messages

This message auto-hides after 3 seconds.

###  Completed Popup  
- A popup appears when user completes the last task then autohides after 4 seconds


###  Timestamps  
- Tasks have a date added in short form.

###  Local Storage Persistence  
- Tasks stay put even after refreshing.
